Getting Started

Installation

Join Pro is distributed as an Autodesk App Store add-in. Two installer options are provided:

Installer Type When to Use Admin Rights Required
Single-User Personal Revit installation on your computer No
Multi-User Corporate deployment; install for all users on a machine Yes

Installation Steps

  1. Download the appropriate installer (.msi file) from Autodesk App Store
  2. Double-click the installer and follow the on-screen prompts
  3. Restart Autodesk Revit completely (close all instances)
  4. Open Revit ‚Üí Look for "Join Pro" button in the Manage tab of the ribbon
  5. Click the button to open the Join Pro palette

How to Use Join Pro

Step 1: Open the Palette

Click the Join Pro button in the Manage tab of the Revit ribbon. The Join Pro palette appears on the right side of your screen.

Step 2: Arrange Priority Order

Join Pro comes with a default priority order:

1. Structural Foundations
2. Walls
3. Columns
4. Structural Framing (Beams)
5. Floors (Slabs)
6. Roofs

To change priority:

  • Click any element in the list to select it
  • Click ‚¨Ü button (green, on the right) to move it up (higher priority)
  • Click ‚¨á button (red, on the right) to move it down (lower priority)
  • Higher position = higher priority = will cut lower-priority elements
üí° Tip: Use the ‚Üª Reset to Defaults button to restore the original priority order at any time.

Step 3: Select Elements in Your Model

In Revit, select the structural elements you want to join:

  • Click elements in the 3D view to select them
  • Hold Ctrl to add to selection
  • Hold Shift to select a range
  • Use Tab to cycle through overlapping elements
  • Use window selection to select multiple elements at once
‚ö† Important: Make sure selected elements are in the same 3D view. Elements from different views or linked models cannot be joined.

Step 4: Execute Join Operation

Click "JOIN BY PRIORITY" to join all selected elements according to the priority order you set:

  • Higher-priority elements will cut through lower-priority elements
  • Multiple joins are processed automatically
  • The operation is undoable (Ctrl+Z)

Join Pro will:

  1. Group selected elements by their category (walls, columns, etc.)
  2. Detect intersections between elements in different groups
  3. Join intersecting elements
  4. Set join order so higher-priority elements cut lower-priority ones
  5. Display results in the details panel

Other Functions

Unjoin All Selected

Click "Unjoin All Selected" to separate all joined elements in your selection from any other elements:

  • The selected elements are unjoined from everything
  • Elements not selected are unaffected
  • Useful for undoing previous joins or starting fresh

Disallow Beam Joins

Click "Disallow Beam Joins" to prevent beam end connections from being automatically joined:

  • Disallows joins at beam ends (critical for MEP/structural separation)
  • Applied to all structural framing elements in your selection
  • Prevents accidental joins with other geometry

Best Practices

  • Test on a copy first: Before running joins on a large model, test on a copy or a small section
  • Work in 3D view: Joins are computed based on 3D geometry, so use 3D views for best results
  • One view at a time: Join elements that are in the same 3D view; avoid mixing linked models
  • Use Undo liberally: If results don't look right, press Ctrl+Z to revert
  • Check intersections: Join Pro only joins elements that have overlapping bounding boxes
  • Verify manually: After joining, review the model to ensure joins are correct

Known Limitations

  • Same view only: Can only join elements in the active 3D view
  • No linked models: Cannot join elements from different linked Revit files
  • Bounding box detection: Elements must have overlapping bounding boxes to be detected for joining
  • Special geometry: Highly curved or complex geometry may not join automatically
  • System families: Some system family elements may have restrictions on joining (Revit limitations)

Troubleshooting

‚ùå "No elements selected!" error

Solution: Select at least one element in the Revit 3D view before clicking any Join Pro button.

‚ùå No joins occur when clicking "JOIN BY PRIORITY"

Possible causes:

  • Selected elements don't physically intersect (check with Ctrl+drag bounding box)
  • Elements are in different views (switch to the correct 3D view)
  • Elements are already joined (use "Unjoin All Selected" first)
  • Elements are not joinable (check Revit documentation for your element types)

Solution: Check the details panel for specific error messages. Verify elements visually in 3D.

‚ùå "Unjoin All Selected" doesn't work with multiple elements

Solution: This has been fixed in the latest version. Update Join Pro from the Autodesk App Store.

‚ùå Join Pro button doesn't appear in Revit

Possible causes:

  • Installation failed (reinstall the .msi)
  • Revit was not fully restarted after installation
  • Running an incompatible Revit version (Join Pro supports 2021‚Äì2026)

Solution: Reinstall Join Pro. Ensure Revit is completely closed before installing.

Frequently Asked Questions (FAQ)

Q: Can I join elements from different linked Revit files?

A: No. Join Pro only works within a single Revit file. Elements from linked files must be joined using Revit's native join tools.

Q: Is Join Pro free?

A: Yes! Join Pro is available for free download from the Autodesk App Store.

Q: Will Join Pro work with Revit 2020 or earlier?

A: No. Join Pro supports Revit 2021 and later. For older versions, use Revit's native join tools.

Q: Can I use Join Pro in Revit for Mac?

A: No. Join Pro is currently Windows only (Revit for Mac has different API constraints).

Q: Can I undo Join Pro operations?

A: Yes! All operations are fully reversible with Ctrl+Z (undo) or the Edit menu.

Q: Does Join Pro modify my project file automatically?

A: No. Join Pro performs operations within a transaction. You must save your project manually to persist changes. Use Ctrl+S or File ‚Üí Save.
