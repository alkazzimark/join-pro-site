Privacy Policy

Join Pro ("Application," "we," "our," or "us") is committed to protecting your privacy. This Privacy Policy explains our data practices for the Join Pro Revit add-in application.

1. Data We Collect

Join Pro does NOT collect any personal data from users. The Application:

  • Does NOT track user activity or usage patterns
  • Does NOT collect email addresses, names, or identifying information
  • Does NOT send any data to external servers or third parties
  • Does NOT use analytics, tracking cookies, or telemetry tools
  • Does NOT store configuration or preferences on remote servers
  • Operates entirely within your local Revit installation on your computer

2. How Data Is Used

Join Pro only processes Revit model data locally on your computer during the session. All processing occurs in-memory:

  • Selected elements are analyzed in real-time to determine join operations
  • Bounding boxes are calculated temporarily to detect element intersections
  • All operations are performed using Autodesk Revit's native JoinGeometryUtils API
  • No data is transmitted, stored persistently, or shared outside your local machine

3. Data Retention & Deletion

Because Join Pro does not collect or store any personal data, there is nothing to retain or delete. All temporary processing data is cleared when you:

  • Close the Join Pro palette
  • Close Revit
  • Clear your Revit project's undo history

Join Pro operations are fully reversible through Revit's native Undo function.

4. User Consent & Data Requests

Because Join Pro does not collect personal data:

  • No consent is required to use the Application
  • No data deletion requests are necessary – no personal information exists
  • No data portability is applicable – there is no user data to export

5. Third-Party Services & SDKs

Join Pro does not use any third-party analytics, advertising networks, or SDKs that would collect user data. The Application only depends on:

  • Autodesk Revit API: The native application programming interface provided by Autodesk for Revit add-in development
  • .NET Framework: Microsoft's runtime used to execute the Application

Neither Autodesk nor Microsoft can identify individual Join Pro users through our Application.

6. Security

Join Pro operates within the Autodesk Revit sandbox environment, which provides inherent security protections:

  • The Application runs with limited permissions within Revit's security context
  • Join Pro cannot access files, network resources, or user data outside of Revit
  • All join/undo operations are managed through Revit's transaction system
  • Changes can be rolled back at any time using Revit's undo feature
Note: Your Revit project files and model data remain entirely under your control and are stored locally on your computer or your organization's servers.

Continued use of Join Pro after such changes constitutes acceptance of the updated Privacy Policy.
